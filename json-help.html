‚ö†Ô∏è The Dreaded ".txt" Trap ‚ö†Ô∏è

Computers try to be "helpful" by secretly adding .txt to the end of your file name. If your file is saved as my_quiz.json.txt, our website will reject it!

How to beat the trap:

  • Windows: At the bottom of the save window, change "Save as type" from Text Documents (*.txt) to All Files (*.*).
  • Mac: Uncheck the little box that says "If no extension is provided, use .txt".
  • Name your file and type .json at the very end (Example: animals.json).
  1. Go back to the Gapfill app and click the Load File button to upload your new file!
